#  Vehicle Rental System (Java + JDBC + MySQL)

A console-based Vehicle Rental System built using **Java, JDBC, and MySQL**.  
This project manages vehicle records and supports full CRUD operations along with rental functionality.



## 📌 Features:
-  Add new vehicles
-  View all vehicles
-  Search vehicle by ID
-  Update vehicle rent price
-  Delete vehicle record
-  Rent a vehicle (mark as unavailable)
-  Return vehicle (mark as available)


## 🛠️ Technologies Used :
- Java (Core Java, OOP)
- JDBC (Database Connectivity)
- MySQL (Database)
- Scanner (User Input)
  

## 🗄️ Database Table
Table: `vehiclest`
